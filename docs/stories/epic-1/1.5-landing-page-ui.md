# Story 1.5: Landing Page UI

## Status
Done

---

## Story

**As a** prospective user,
**I want** to see a clean, modern landing page that showcases the AI video generation platform's value,
**so that** I can understand the product benefits and be encouraged to sign up or log in.

---

## Acceptance Criteria

1. Landing page displays at route `/` with a full-screen hero section featuring the platform headline and value proposition
2. Hero section includes a primary yellow CTA button ("Get Started Free" or "Login") prominently displayed
3. Navigation bar appears at the top with logo, feature links, pricing link, and login button (top-right)
4. Features section displays 3 key product benefits in a grid layout with glass cards and icons
5. Social proof section includes testimonial or trust signal (e.g., "Join 1000+ businesses creating ads...")
6. All UI components follow glassmorphism design system (background blur, semi-transparent white borders, rounded corners)
7. Color scheme matches brand identity: Lightning Yellow (#FFEB3B) for CTAs, Cosmic Dark (#0A0E27) for backgrounds, White for text
8. Page is responsive and optimized for desktop-first design (mobile support secondary)
9. Clicking "Login" button navigates to `/login` page
10. Clicking "Get Started" / primary CTA navigates to `/signup` page
11. All icons use Lucide React icon library
12. Page loads within 2 seconds and achieves passing Lighthouse performance score (>80)

**Colors:**
- Lightning Yellow (Primary CTA): `#FFEB3B`
- Cursor White (Text): `#FFFFFF`
- Cosmic Dark (Background): `#0A0E27`
- Mid Navy (Secondary bg): `#1A2332`
- Light Blue (Accent): `#4FC3F7`

**Typography:**
- Font Stack: `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial`
- Hero Headline: 2.25rem-3.75rem bold
- Body text: 1rem-1.125rem regular
- All text should be white or white with opacity

**Glassmorphism CSS:**
```css
.glass-card {
  background: rgba(255, 255, 255, 0.05);
  backdrop-filter: blur(10px);
  border: 1px solid rgba(255, 255, 255, 0.1);
  border-radius: 12px;
  box-shadow: 0 8px 32px 0 rgba(0, 0, 0, 0.37);
}

.btn-primary {
  background: linear-gradient(135deg, #FFEB3B 0%, #FFC107 100%);

  font-weight: 600;
  border-radius: 8px;
  padding: 12px 24px;
  box-shadow: 0 4px 16px rgba(255, 235, 59, 0.3);
}
```
